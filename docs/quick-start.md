# 快速开始

## 编译文档

### 1. 本地编译

- **编译论文** `thesis.pdf`：

   ```bash
   latexmk -xelatex thesis.tex
   ```

- **清理编译临时文件**：

   ```bash
   latexmk -c
   ```

### 2. Overleaf 编译

直接使用 [Overleaf 模板](https://www.overleaf.com/latex/templates/10290-zhong-guo-kuang-ye-da-xue-shuo-shi-sheng-bi-ye-lun-wen-mo-ban/qwggynbswxwg) 进行在线编译。（近期将同步更新）
